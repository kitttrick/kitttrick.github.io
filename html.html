Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<caption> Caption The title shown above the table.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Footer The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
class Class Specifies one or more classnames for an element, for use in a style sheet.
id ID Specifies a unique id for an element
hidden Hidden Specifies that an element is not yet, or is no longer, relevant.
style Style Specifies an inline CSS style for an element.
title Title Specifies a tooltip when the element is hovered over.
