Commands

Command Description
git log Prints commit logs
git status Displays branch name, shows if branch is up to date with main, shows any uncommitted changes
git init Initialises git repo
mkdir NAME Makes a folder in the current location called NAME
touch NAME.txt Makes a file in the current location called NAME
git branch BRANCHNAME Creates a new branch called BRANCHNAME
git add example.txt Adds example.txt to staging area
git commit -m "example message" Commits with a message of "example message"
git merge BRANCHNAME Merges BRANCHNAME into main
git checkout BRANCHNAME Switches to branch called BRANCHNAME
git branch -a Lists all branch names
git branch -d BRANCHNAME Deletes branch called BRANCHNAME
git diff Shows all uncommitted changes
git rm example.txt Removes example.txt from staging area
git push Push changes to remote upstream
git pull Pulls changes from main
git push --set-upstream origin BRANCHNAME Set upstream of branch
git branch --unset-upstream Unset upstream of current branch
git stash Stash changes
git stash pop Unstash changes
git clone URL Clones the repo at the URL locally
git remote -v View all remotes of project
git fetch Fetches all changes to the remote, but does not merge them
